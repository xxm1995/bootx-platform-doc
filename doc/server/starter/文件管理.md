# 文件管理
## 功能
- [x] 服务器本地文件存储  
- [x] MongoDB存储
- [ ] Minio
- [ ] 阿里云OSS
- [ ] 腾讯COS
## 使用
> 见 `FileUploadService`、`UploadService`
### 本地存储
将文件上传到应用所在服务器上，需要配置存储路径，默认为`/data/file/`
### Mongo存储
将上传的文件存储到`MongoDB`中，基于`GridFsTemplate`进行开发，需要配置`bucket`存储桶参数

### 配置文件
```yaml
# 项目配置
bootx.file-upload:
    # 文件存储方式
    upload-type: LOCAL
    mongo:
      # 存储桶
      bucket: fs
    local:
      # 本地存储路径
      local-path:
    # 文件服务器地址
    server-url: http://127.0.0.1:8080
```
